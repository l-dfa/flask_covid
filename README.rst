prerequisites
---------------

* python >= 3.6

install
--------------

In cmd::

  cd flask_covid
  python -m venv venv
  venv\Scripts\activate   # or venv/bin/activate on Linux
  pip install -r requirements.txt
  
Exec
-----------

In cmd::

  cd flask_covid
  venv\Scripts\activate   # or venv/bin/activate on Linux
  flask run
  
please, use a web browser to show http://localhost:5000
